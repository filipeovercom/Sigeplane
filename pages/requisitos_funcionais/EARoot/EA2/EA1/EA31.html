Como administrador do sistema desejo gerenciar o cadastro de usuários. Desejo cadastrar, alterar e desativar usuários.<br/>

Requirement Type Status Difficulty Priority Stability
Alterar Informações do Usuário Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "Alterar Informações do Usuário" o sistema deve exibir uma janela com o formulário para alteração dos dados:<br/><ul>
<li>Titulação</li><li>Nome</li><li>Matrícula</li><li>E-mail</li><li>Perfis</li></ul><p/>
Cadastrar Usuário Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "<b>Cadastrar novo Usuário</b>" o sistema deve exibir uma janela com um formulário contendo os seguintes campos:<br/><ul>
<li>Perfil [Administrador, Coordenador...]</li><li>Titulação [Me, Esp, Dr][Caso o perfil escolhido seja Coordenador ou Professor]</li><li>Nome [Nome completo]</li><li>Matrícula</li><li>E-mail</li><li>Senha</li></ul><p/>
Desativar Usuário Functional Proposed Medium Medium Moderate
Details:
A listagem de usuários deve possuir a ação "<b>Desativar Usuário</b>". Um usuário que foi desativado não poderá mais realizar autenticação no sistema. <br/>
Listar Usuários Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma tela que exibe os usuários cadastrados e disponiblizar acesso as ações:<br/><ul>
<li>Cadastrar Usuário</li><li>Alterar Informações do Usuário</li><li>Desativar Usuário</li></ul><p/>
Constraint Type Status
Múltiplos Perfis Process Approved
Details:
Um usuário poderá acumular perfis. A aplicação deve se adaptar aos perfis de acesso que o usuário possui.<br/>
Cadastrar/Alterar Professor Process Approved
Details:
Ao cadastrar ou alterar um usuário é obrigatório o preenchimento do nome, matrícula e perfil(is). A senha é opcional, pois o sistema irá realizar a autenticação com a rede da instituição utilizando a senha informada no momento do login.<br/>
Scenario Type
Listar Usuários Basic Path
Details:
Steps:
1. Aciona o menu Usuários<br/>
2. Redireciona para tela de listagem de Usuários<br/>
3. Verifica se usuário possui perfil Administrador<br/>
4. Exibe listagem dos usuários cadastrados no sistema<br/>
Cadastrar Usuário Basic Path
Details:
Steps:
1. Aciona a ação Cadastrar novo Usuário<br/>
2. Exibe janela com formulário para cadastro<br/>
3. Preenche formulário e aciona ação salvar<br/>
4. Valida informações e registra Usuário na base de dados<br/>
Alterar Informações do Usuário Basic Path
Details:
Steps:
1. Aciona menu Alterar Informações do Usuário<br/>
2. Exibe janela com formulário para alteração dos dados<br/>
3. Altera informações e aciona ação Salvar<br/>
4. Valida informações e registra a alteração na base de dados<br/>
Desativar Usuário Basic Path
Details:
Steps:
1. Aciona a ação Desativar Usuário<br/>
2. Exibe janela de confirmação<br/>
3. Confirma desativação<br/>
4. Desativa o Usuário na base de dados<br/>
