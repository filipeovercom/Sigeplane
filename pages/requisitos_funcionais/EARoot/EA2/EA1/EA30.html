Os alunos, professores e coordenadores necessitam realizar a autenticação no sistema para poder ter acesso aos devidos recursos reservados para seus perfis.<br/><br/>O sistema possui os seguintes perfis:<br/><ul>
<li>Administrador</li><li>Coordenador</li><li>Professor</li><li>Aluno</li></ul>
<br/>Para realizar autenticação no sistema o usuário deverá informar matrícula e senha que utiliza para autenticação na rede da instituição.<br/>

Requirement Type Status Difficulty Priority Stability
Usuário Administrador Functional Proposed Medium Medium Moderate
Details:
O usuário com o perfil Administrador tem acesso a todos os recursos administrativos do sistema como cadastros básicos e gerenciamento de usuários.<br/><br/>O usuário que possui somente o perfil administrador não poderá ter acesso ao gerencimento de planos de ensino, que somente é acessível por Coordenadores e Professores.<br/>
Usuário Aluno Functional Proposed Medium Medium Moderate
Details:
O usuário com perfil Aluno somente possui acesso à visualização dos Planos de Ensino.<br/>
Usuário Coordenador Functional Proposed Medium Medium Moderate
Details:
O usuário com o perfil Coordenador possui acesso ao gerenciamento de cadastros básicos e gerenciamento de planos de ensino onde poderá aprovar ou reprovar os planos de ensino do curso.<br/>
Usuário Professor Functional Proposed Medium Medium Moderate
Details:
O usuário com o perfil Professor possui acesso ao Gerenciamento de Planos de Ensino onde poderá produzir os planos de ensino das turmas em que o mesmo é responsável.<br/><br/>O usuário Professor não possui acesso ao gerenciamento de cadastros básicos e gerenciamento de usuários.<br/>
