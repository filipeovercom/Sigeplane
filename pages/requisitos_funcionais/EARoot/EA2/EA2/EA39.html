Como Administrador ou Coordenador desejo cadastrar Competências para serem utilizadas como sugestão no preenchimento do plano de ensino pelo professor.<br/><br/>Em um plano de ensino um dos objetivos é formar competências nos alunos. São exemplos de competências:<br/><br/><ul>
<li>Compreender as ciências como construções humanas, relacionando o desenvolvimento científico ao longo da história com a transformação da sociedade.</li></ul>
<br/><ul>
<li>Compreender o papel das ciências naturais e das tecnologias a elas associadas, nos processos de produção e no desenvolvimento econômico e social contemporâneo.</li></ul>
<br/>Uma competência pode ser trabalhada por um ou mais cursos em uma ou mais disciplinas.<br/>

Alterar Informações da Competência Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "Alterar Competência" na tela de listagem de competências o sistema deve exibir uma janela com um formulário para alteração das informações da competência selecionada. O formulário deve conter os seguintes campos:<br/><ul>
<li>Descrição da Competência</li><li>Curso(s) relacionado(s)</li></ul><p/>
Cadastrar Competências Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "Cadastrar nova Competência" o sistema deve exibir uma janela com um formulário para cadastro contendo os seguintes campos:<br/><ul>
<li>Descrição</li></ul>
<ul>
<li>Curso(s) [Selecionar da tabela de cursos]</li></ul><p/>
Desativar Competências Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma função para desativar competências que não serão mais utilizadas.<br/>Ao desativar uma competência o sistema não a sugere mais no preenchimento do plano de ensino.<br/>
Listagem de Competências Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma tela que exibe as competências cadastradas. <br/><br/>Esta tela deve disponibilizar acesso às funções de cadastrar, editar e desativar uma competência.<br/><br/>Nesta tela o usuário poderá consultar as competências por descrição ou nome do curso.<br/><br/>Esta tela deve possuir uma tabela com as seguintes colunas:<br/><ul>
<li>Descrição da competência</li><li>Nome do(s) Curso(s)</li></ul>
<br/>OBS: O curso somente é relacionado a competência para tornar a visualização e procura mais produtiva, não havendo nenhum relacionamento que restringe sua usabilidade.<br/><br/>
Constraint Type Status
Controle de Acesso Pre-condition Approved
Details:
Para acessar as funcionalidades o usuário deve possuir o perfil Administrador ou Coordenador<br/>
Cadastrar/Alterar Competência Process Approved
Details:
Ao cadastrar ou alterar uma competência o sistema deve exigir o preenchimento da descrição.<br/>O sistema deve exibir competências similares com a descrição que é informada pelo usuário. Antes de salvar o sistema deve exigir uma confirmação e ao mesmo tempo exibir as competências similares. O objetivo desta ação é diminuir a quantidade de dados repetidos.<br/><br/>
Scenario Type
Listar Competências Basic Path
Details:
Steps:
1. Aciona menu Competências<br/>
2. Redireciona usuário para tela de listagem de competências<br/>
3. Verifica Permissão de Acesso<br/>
4. Exibe listagem de competências<br/>
Cadastrar Competência Basic Path
Details:
Steps:
1. Aciona ação Cadastrar Nova Competência na tela de listagem de competências<br/>
2. Exibe janela com formulário para preenchimento das informações<br/>
3. Preenche as informações e clica em salvar<br/>
4. Valida informações e registra a competência na base de dados<br/>
Alterar Informações da Competência Basic Path
Details:
Steps:
1. Seleciona uma competência na listagem e aciona a ação Alterar Competência<br/>
2. Exibe formulário para alteração das informações da competência<br/>
3. Altera as informações e clica em salvar<br/>
4. Valida as informações e altera o registro na base de dados<br/>
Desativar Competência Basic Path
Details:
Steps:
1. Seleciona uma ou várias competências na listagem e aciona a ação Desativar<br/>
2. Exibe janeja pedindo confirmação da ação<br/>
3. Confirma ação e clica em desativar<br/>
4. Desativa as competências na base de dados<br/>
