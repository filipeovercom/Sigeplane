Como Administrador ou Coordenador desejo cadastrar Metodologias para serem utilizados como sugestão no preenchimento do plano de ensino pelo professor.<br/><br/>Para que o aluno obtenha uma habilidade é nessário que o professor ministre os conteúdos adequados e da forma adequada. A forma como os conteúdos serão ministrados é chamada de metodologia.<br/><br/>São exemplos de metodologias:<br/><ul>
<li>Aulas expositivas dialogadas com a participação ativa dos alunos.</li></ul>
<ul>
<li>Resolução de exercícios individuais, ou em grupos, em sala de aula/ambiente virtual.</li><li>Trabalhos/pesquisas extraclasse (Estudo Dirigido para análise de publicações periódicas<b>).</b></li></ul>

Requirement Type Status Difficulty Priority Stability
Alteração de uma metodologia Functional Proposed Medium Medium Moderate
Details:
A tela de listagem deve possuir uma opção para alterar uma metodologia.<br/><br/>Ao acionar a ação o sistema deve exibir um formulário para alteração das informações:<br/><br/><ul>
<li>Nome</li><li>Descrição</li></ul><p/>
Cadastro de nova metodologia Functional Proposed Medium Medium Moderate
Details:
Na tela de listagem deve existir a opção para cadastrar novas metodologias.<br/><br/>Ao acionar esta ação o sistema deve exibir um formulário com os seguintes campos:<br/><br/><ul>
<li>Nome da Metodologia</li><li>Descrição da metodologia</li></ul><p/>
Exclusão de Metodologias Functional Proposed Medium Medium Moderate
Details:
A tela de listagem de metodologias deve possuir a opção de exclusão de uma metodologia.<br/><br/>Ao acionar esta ação o sistema deve exigir confirmação por parte do usuário. Ao confirmar a realização da ação o sistema deve eliminar a metodologia da base de dados.<br/>
Listagem de metodologias Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma tela para listagem de todas as metodologias cadastradas na base de dados.<br/><br/>Nesta tela deve ser exibido também as opçãos de cadastro, alteração e exclusão de metodologias. As metodologias devem ser exibidas em uma tabela com as seguintes colunas:<br/><ul>
<li>Nome</li><li>Descrição</li><li>Dados de Utilização</li></ul><p/>
Constraint Type Status
Controle de Acesso Pre-condition Approved
Details:
Para ter acesso a esta funcionalidade o usuário deve possuir o perfil Administrador e/ou Coordenador.<br/>
Validação do Cadastro/Alteração Post-condition Approved
Details:
Para cadastro e alteração de uma metodologia é obrigatório a informação do nome da mesma.<br/>
Scenario Type
Listagem de Metodologias Basic Path
Details:
Steps:
1. Aciona o menu Metodologias na seção Gerenciamento de Cadastros<br/>
2. Redireciona o usuário para a tela de listagem de metodologias<br/>
3. Verifica as permissões de acesso<br/>
4. Exibe as metodologias cadastradas no sistema<br/>
Exceptions:
4a. Cadastro de Metodologia (at End)
4b. Alteração de Metodologia (at End)
4c. Exclusão de Metodologia (at End)
Cadastro de Metodologia Alternate
Details:
Steps:
1. Aciona a ação Cadastro de Nova Metodologia<br/>
2. Exibe formulário para preenchimento das informações<br/>
3. Preenche as informações e clica em salvar<br/>
4. Valida as informações e salva o registro na base de dados<br/>
Alteração de Metodologia Alternate
Details:
Steps:
1. Aciona a ação Alterar Metodologia<br/>
2. Exibe formulário para alteração das informações<br/>
3. Altera as informações desejadas e clica em salvar<br/>
4. Valida as informações e registra a alteração na base de dados<br/>
Exclusão de Metodologia Alternate
Details:
Steps:
1. Aciona a ação Excluir Metodologia<br/>
2. Exibe janela de confirmação<br/>
3. Confirma a ação<br/>
4. Elimina a metodologia da base de dados<br/>
