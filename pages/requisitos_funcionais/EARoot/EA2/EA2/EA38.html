Appears In: Requisitos Funcionais
O sistema deve disponibilizar um gerenciamento das disciplinas ofertadas pelos cursos da instituição.<br/><br/>As disciplinas são utilizadas para formar as turmas e distinguir cadastros.<br/><br/>As disciplinas podem ou não ser agrupdas por curso. Para acessar o gerencimaento de disciplinas o usuário deve possuir o perfil Administrador.<br/>

Requirement Type Status Difficulty Priority Stability
Alteração de Disciplinas Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "Alterar Disciplina" o sistema deve exibir uma janela com um formulário para alterar os dados de uma disciplina.<br/><br/>O usuário poderá alterar: <br/><ul>
<li>Código Interno</li><li>Nome</li><li>Nome Reduzido</li><li>Curso</li><li>Ementa</li><li>Contribuição para formação</li><li>Bibliografia</li></ul><p/>
Cadastro de Disciplinas Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "Cadastrar nova Disciplina" o sistema deve exibir uma janela com um formulário para cadastro com os seguintes campos:<br/><br/><ul>
<li><b>Código Interno</b></li><li><b>Nome</b> [Ex: Arquitetura de Computadores I]</li></ul>
<ul>
<li><b>Nome Reduzido</b> [Ex: Arq. Comp. I]</li><li><b>Curso</b> [Selecionar da tabela de cursos]</li><li><b>Ementa</b></li><li><b>Carga horária Prática</b></li><li><b>Carga horária Teórica</b></li></ul>
<ul>
<li><b>Ementa</b></li><li><b>Contribuição para formação</b></li><li><b>Bibliografia</b></li></ul><p/>
Desativação/Exclusão de Disciplinas Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma funcionalidade onde um usuário devidamente autorizado poderá desativar/excluir disciplinas.<br/><br/>Esta funcionalidade deve estar acessível na tela de listagem de disciplinas.<br/><br/>Caso a disciplina não esteja vinculada a nenhuma turma a mesma deve ser removida. Caso contrário somente será desativada, podendo ser reativa posteriormente.<br/><br/>
Listagem de Disciplinas Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma tela que exibe as disciplinas cadastradas.<br/><br/>Esta tela deve disponibilizar acesso às funções de cadastrar, editar e desativar uma disciplina.<br/><br/>Nesta tela o usuário poderá consultar as disciplinas por código interno, nome, nome reduzido e nome do curso.<br/><br/>Esta tela deve possuir uma tabela com as seguintes colunas:<br/><ul>
<li><b>Código Interno</b>. Ex: SI001</li><li><b>Nome</b>. Ex: Arquitetura de Computadores I</li><li><b>Nome Reduzido</b>. Ex: Arq. de Comp. I</li><li><b>Ativa</b>. Ex: Sim ou Não</li></ul><p/>
Constraint Type Status
Controle de Acesso Pre-condition Approved
Details:
Somente os usuários com o perfil Administrador e/ou Coordenador podem acessar esta tela.<br/>
Cadastro e Alteração Process Approved
Details:
O sistema deve exigir um nome para a disciplina, não permitindo o usuário salvar uma disciplina sem nome.<br/>
Desativar/Excluir Disciplina Process Approved
Details:
Não é possível excluir uma disciplina caso a mesma esteja vinculada a uma turma. Caso a mesma esteja vinculada somente é possível desativá-la.<br/><br/>Ao desativar uma disciplina:<br/><ul>
<li>Não é possível relacionar a disciplina desativa com uma turma.</li><li>Os planos de ensino de todas as turmas relacionadas a disciplina desativa ficam inacessíveis.</li></ul><p/>
Scenario Type
Listagem de Disciplinas Basic Path
Details:
Steps:
1. Acessa o menu Disciplinas na seção de Gerenciamento de Cadastros<br/>
2. Verifica permissão de acesso<br/>
3. Exibe as disciplinas cadastradas no sistema<br/>
Exceptions:
3a. Cadastro de disciplina (at End)
3b. Alteração de disciplina (at End)
3c. Desativação/Exclusão de disciplina (at End)
Cadastrar Nova Disciplina Basic Path
Details:
Steps:
1. Aciona ação Cadastrar nova Disciplina<br/>
2. Exibe janela com formulário para cadastro<br/>
3. Preenche o formulário e aciona ação Salvar<br/>
4. Valida informações e registra a disciplina na base de dados<br/>
Alterar Disciplina Basic Path
Details:
Steps:
1. Aciona ação Alterar Disciplina<br/>
2. Exibe formulário para alteração da disciplina<br/>
3. Altera informações e aciona ação Salvar<br/>
4. Valida informações e registra alteração na base de dados<br/>
Desativar Disciplina Basic Path
Details:
Steps:
1. Aciona ação Desativar Disciplina<br/>
2. Exibe janela de confirmação<br/>
3. Confirma desativação<br/>
4. Desativa a disciplina<br/>
Cadastro de disciplina Alternate
Details:
Steps:
1. Aciona o menu de cadastro de disciplina na tela de listagem<br/>
2. Exibe o formulário para cadastro da disciplina<br/>
3. Preenche as informações e clica em salvar<br/>
4. Valida as informações e salva o registro na base de dados<br/>
Alteração de disciplina Alternate
Details:
Steps:
1. Aciona o menu de Alteração de uma disciplina<br/>
2. Exibe formulário para alteração das informações<br/>
3. Altera as informações desejadas e clica em salvar<br/>
4. Valida as informações e registra a alteração na base de dados<br/>
Desativação/Exclusão de disciplina Alternate
Details:
Steps:
1. Aciona a opção de desativação/exclusão<br/>
2. Exibe janeja para confirmação da ação<br/>
3. Confirma a desativação/exclusão da disciplina<br/>
4. Desativa ou Exclui a disciplina<br/>
