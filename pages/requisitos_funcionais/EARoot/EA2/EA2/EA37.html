Como coordenador desejo gerenciar o cadastro de turmas do semestre letivo. O cadastro de turmas relaciona:<br/><br/><ul>
<li>Semestre</li><li>Curso</li><li>Disciplina</li><li>Professor</li></ul><p/>

Alterar Informações da Turma Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "<b>Alteração da Turma</b>", na tela de listagem de turmas, o sistema deve exibir ao usuário uma janela com um formulário para alteração dos dados da turma selecionada, exibindo os seguintes campos:<br/><br/><ul>
<li>Código Interno da Turma [Campo Texto]</li><li>Curso [Selecionar da Tabela de Cursos]</li><li>Disciplina [Selecionar da Tabela de Disciplinas]</li><li>Professor [Selecionar da Tabela de Usuários(Professores)]</li></ul><p/>
Cadastrar Nova Turma Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "<b>Cadastro de Nova Turma</b>", na tela de listagem de turmas, o sistema deve exibir ao usuário uma janela com um formulário para cadastro de uma nova turma, exibindo os seguintes campos:<br/><br/><ul>
<li>Código Interno da Turma [Campo Texto]</li><li>Curso [Selecionar da Tabela de Cursos]</li><li>Disciplina [Selecionar da Tabela de Disciplinas]</li><li>Professor [Selecionar da Tabela de Usuários(Professores)]</li></ul><p/>
Desativar Turma(s) Functional Proposed Medium Medium Moderate
Details:
Na tela de listagem de turmas deve existir uma opção para desativar uma ou mais turmas. <br/><br/>As turmas desativadas não podem ser visualizadas pelos professores ou alunos.<br/>
Listar Turmas Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma tela que exiba as turmas cadastradas no semestre ativo assim como disponiblizar acesso as ações:<br/><ul>
<li>Cadastrar Turma</li><li>Alterar Turma</li><li>Desativar Turma</li></ul><p/>
Constraint Type Status
Controle de Acesso Invariant Approved
Details:
Somente os usuários com perfil "<b><u>Coordenador</u></b>" podem acessar as funcionalidades de gerenciamento das turmas.<br/>
Cadastrar Nova Turma Process Approved
Details:
No cadastro de nova turma a escolha do professor é opcional, pois é comum cadastrar a turma e depois decidir qual professor será responsável por ela. Os demais campos são obrigatórios.<br/>
Alterar Informações da Turma Process Approved
Details:
Não é possível alterar informações da turma caso exista um plano de ensino vinculado a mesma.<br/>
Scenario Type
Visualizar Turmas Basic Path
Details:
Steps:
1. Aciona menu Turmas<br/>
2. Redireciona usuário para tela de listagem de turmas<br/>
3. Verifica se usuário possui perfil Coordenador<br/>
4. Exibe as turmas cadastradas de acordo com o contexto [Semestre]<br/>
Cadastrar Nova Turma Basic Path
Details:
Steps:
1. Aciona ação "Cadastrar Nova Turma"<br/>
2. Exibe formulário de cadastro da nova turma<br/>
3. Preenche formulário e aciona ação para salvar<br/>
4. Registra a turma na base de dados e exibe mensagem para o usuário<br/>
Alterar Informações da Turma Basic Path
Details:
Steps:
1. Aciona a ação "Alterar Turma"<br/>
2. Exibe janela com formulário para alteração das informações da turma<br/>
3. Altera as informações desejadas e aciona a ação Salvar<br/>
4. Altera as informações da turma na base de dados e exibe mensagem para o usuário<br/>
Desativar Turma Basic Path
Details:
Steps:
1. Seleciona a(s) turma(s) que deseja desativar<br/>
2. Acionar a ação "Desativar Turma"<br/>
3. Exibe janela de confirmação<br/>
4. Confirma desativação<br/>
5. Desativa turma na base de dados<br/>
