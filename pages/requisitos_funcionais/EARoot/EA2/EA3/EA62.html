Como aluno da instituição de ensino desejo visualizar o plano de ensino da instituição quando o mesmo estiver disponível.<br/>

Requirement Type Status Difficulty Priority Stability
Escolha da Turma Functional Proposed Medium Medium Moderate
Details:
O sistema deve possuir uma tela para que o usuário possa pesquisar pelo plano de ensino que deseja visualizar.<br/><br/>A tela deve oferecer os seguintes filtros:<br/><ul>
<li>Semestre [Por padrão mostrar o semestre ativo]</li><li>Curso</li><li>Disciplina</li></ul>
<br/>Ao realizar a busca o sistema deve exibir as turmas encontradas e disponibilizar ao usuário a ação "Visualizar Plano de Ensino".<br/>
Visualização do Plano de Ensino Functional Proposed Medium Medium Moderate
Details:
Ao acionar a ação "Visualizar Plano de Ensino" o sistema deve abrir em nova tela/janela a visualização do documento, disponibilizando a opção de Download.<br/>
Constraint Type Status
Autenticação Pre-condition Approved
Details:
Para o aluno acessar a visualização do plano de ensino é necessário realizar autenticação utilizando a matrícula e senha da rede da instituição.<br/>
Scenario Type
Visualização do Plano de Ensino Basic Path
Details:
Steps:
1. Informa matrícula e senha para autenticação<br/>
2. Valida informações<br/>
3. Redireciona para a tela de busca da turma<br/>
4. Informa os parâmetros desejados<br/>
5. Busca as turmas que correspondem aos parâmetros informados<br/>
6. Exibe uma lista com as turmas encontradas<br/>
7. Aciona a ação "Visualizar Plano de Ensino" da turma que deseja<br/>
8. Abre nova janela com a visualização do documento no formato PDF com opção para download<br/>
